# Webview

## Dev stack
* Node.js
* Electron

## Shortcuts
* [Ctrl + Z]        Navigate back
* [Ctrl + Y]        Navigate forward
* [Ctrl + E]        Quit
* [Ctrl + Space]    Navigate to home page

## How to run
```
npm ci
npm run start
```
## How to build
```
npm ci
npm run make
```
